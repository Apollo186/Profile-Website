Language: C/Python

The final project for my microprocessor lab was to make a gui that can read data from an MSP430FR2355 microcontroller

We had to read the x,y,z directions, noise detection, and what buttons are being pressed

We made the GUI using python in a juptyer notebook, and wrote the the C code in the Energia IDE
